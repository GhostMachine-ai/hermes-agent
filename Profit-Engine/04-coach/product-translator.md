# Product Translator

Converts internal / technical language into customer-benefit language.

## Core Services

| Technical Term | Customer Translation |
|---------------|---------------------|
| Workflow automation | "You stop doing this manually — it just happens." |
| Agent orchestration | "Multiple tasks run in parallel while you focus on one thing." |
| API integration | "Your tools talk to each other — no more copy-paste." |
| LLM-powered processing | "A smart layer reads and sorts everything for you." |
| Data pipeline | "Your numbers are always current, in one place." |
| Prompt engineering | "We write the instructions so the AI does exactly what you need." |
| RAG / retrieval | "It answers questions using your own documents, not generic training." |
| Model fine-tuning | "We teach it your specific language, tone, and rules." |

## Feature → Benefit Translations

### Automated reporting
- **Say**: "You get a clean summary in your inbox every Monday. No one has to build it."
- **Don't say**: "We run a scheduled ETL that aggregates your data into a structured report."

### Custom AI agent
- **Say**: "It handles [specific task] the same way you would, but instantly and at scale."
- **Don't say**: "We deploy a fine-tuned LLM with tool-use capabilities and a ReAct loop."

### Error handling & fallbacks
- **Say**: "If something unexpected happens, it flags it for a human instead of guessing."
- **Don't say**: "We implement exception handlers with human-in-the-loop escalation."
